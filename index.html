How it works

Three steps. No signup. No configuration files. Just paste, upload, and go.

1

Upload your recruiter CSV

Export a candidate list from any ATS — Greenhouse, Lever, Workday — or use our sample CSV to try it out right away. No specific format required.

2

Optionally, add a GitHub profile

Paste in a candidate's GitHub URL and the tool pulls in their real skills, repos, and bio — enriching the profile automatically without any extra work from you.

3

Get clean, structured JSON

We deduplicate, normalize phone numbers, and merge all sources into a single canonical profile. Copy the JSON or browse the visual card view — your choice.
